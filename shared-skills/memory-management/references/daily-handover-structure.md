# Daily Handover Recommended Structure

**Top**
- Synopsis (quick scan + decision rationale)

**Middle**
- Timestamped Compression sections (What / Why / How format preferred)

**Bottom**
- Detailed Work Log
- Open Items / Next Steps

**Flagging**
- Use `**⭐ IMPORTANT**` or `**[!]**` for key nuggets that should be easy to find later.
