# Opus Prompt — Design Bible Card Design

Feed this prompt to Opus/Cursor along with `Bruiser_Design_Bible.md`. Do not feed 10+ separate documents. The Bible is the single source.

```
You are designing Bruiser minion cards for a 1961 Detroit mob TCG.

Read Bruiser_Design_Bible.md completely. It is your single source of 
truth. You will not load other documents unless a specific scoring 
calculation requires the scorer tool.

For each card you design:

1. Start from the WORLD, not the slot. Read the crew scenes in Section 
   2 of the Bible. Find the person in the scene. THEN check what slot 
   needs filling.

2. Follow the Card Design Process (Section 9). Use Path A (need-first) 
   unless Joe gives you a name. Complete the full workup — every block.

3. Propose 3-5 names. Run the name gate yourself (Section 8). Reject 
   any name that fails — do not present failures to Joe.

4. Answer "Who is this person?" and "Why would they do this?" in one 
   sentence each. If the answer is "a generic guy who [keyword]," 
   redesign.

5. Score the card AFTER design. Use the rarity bands in Section 10. 
   Promote rarity if Δ exceeds ceiling. Never assign rarity first.

6. Present ONE card at a time. Card workup table, name options, ludo 
   one-liner, synergy partners, score.

NEVER: "Fill slot #24 with a 4V Street Common Grit body." That prompt 
produces derivative garbage. Start from the world, not the slot.

Design a card from one of the four crew scenes above. Choose any scene. 
Find the person in it. Build the card.
```
