# Worked Example — Source Documents Already Absorbed (B+C)

**Pass type:** Red team briefs + methodology + v3/v4 archives — source documents that fed into live canon.

**Date:** 2026-06-01 · **Batch:** Sections B+C

**Key lesson:** When source documents are the INPUTS that produced the live canon (not parallel versions of it), the Fond Pass disposition is almost entirely FOUND. This is a successful pass — it proves the canon absorbed what the source documents established.

## What we found

Sections B+C contained 14 files: red team briefs v1-v4, methodology docs, v3/v4 alternate roster iterations. Every creative beat — faction identity, crew textures, design rules — was already absorbed into live canon.

## Pattern

1. Read the source — recognize the language as familiar
2. Check live canon — confirm it's there
3. Disposition: FOUND
4. Repeat

The working doc is short. Most beats are FOUND with a single-line cite. No NEW beats. No CONFLICTS.

## How to recognize

- Source files are briefs/framework documents — they tell OTHER agents how to design
- They are NOT parallel versions of the canon — they are the single source that PRODUCED the canon
- Reading them feels like reading the design bible in draft form

## Don't mistake for

- Lore drafts (v2/v3/FINAL with different prose to reconcile)
- Design exercises (alternate card names worth banking)
- Mixed creative (crew texture from alternate design path worth extracting)

Everything here already shipped. The pass proves it shipped.
