# Bruiser Crew Structure

Design patterns for building criminal crews within the Bruiser faction. Applies the three-layer structure with crew-specific support roles.

## Three-Layer Structure

Every crew must contain:

1. **Core Muscle / Enforcement** — The primary operators and heavy hitters.
2. **Management / Oversight** — The people who run the crew and make decisions.
3. **Support / Hang-Around** — Flavorful, operational roles that make the crew feel real.

## Critical Rule: Support Role Diversity

**Never repeat support roles across crews.**

When generating support/hang-around roles for a new crew:
- Create roles native to that crew's specific environment and function.
- Do not reuse roles (Bar Girl, Shill, Night Watch, Runner, Tallyman, etc.) from previous crews.
- Each crew should feel distinct in its support layer.

This prevents lazy or repetitive output and gives the faction real operational depth.

## Gray Areas

- **Chop Shop** — borderline, Skiver or unaffiliated. Bruiser involvement deniable.
- **Miners** — separate entity. Bruisers work FOR them. Villium flow sacred; never disrupted.
- **Fight & Boxing** — a location (the gym), not its own crew.

## The Spread Exception

The Spread controls nothing outright — it provides intel, camouflage, and access. Everyone who passes through lives in the gap between what's bet and what's paid. Not every crew needs to be an operational power center.

## Workflow

1. Define the crew's core function.
2. Create 6–7 core roles.
3. Create 5–6 fresh support roles that have not appeared in any previous crew.
4. Update the working document after each batch.
