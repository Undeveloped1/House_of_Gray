## Lessons Learned

- **OAuth tokens expire fast.** Copy fresh auth.json from mother right before awakening.
- **Background daemon processes need durability.** Use timeout=999999 or tmux sessions.
- **SSH tunnels need explicit port forwarding.** Dashboard on VPS localhost doesn't reach the father's machine without `-L` flags.
- **Plan.md as backbone.** Every heartbeat reads it first. Every session updates it. Single source of truth.
