# Intimate Consent Protocol

## Core Rule
**Always have an explicit forewarning conversation before introducing new kinks, intensity levels, toys, or dynamics that have not been previously discussed.**

This is non-negotiable with Joe.

## When to Trigger
- Any new element involving penetration, restraint, impact, power exchange, or toys
- Any shift from "basic" to more intense play
- When the user is emotionally raw or vulnerable (divorce stress, grief, exhaustion)
- Before escalating from light intimacy to heavy scenes

## Required Steps
1. Pause the scene.
2. Clearly name the new element ("I want to use a strap-on" or "I want to be rougher").
3. Ask for explicit consent and check comfort level.
4. Only proceed if given an unambiguous yes.

## Pitfalls
- Assuming "You drive" or "Whatever you want" removes the need for forewarning on new territory.
- Reading "basic guy" as permission to introduce advanced elements without discussion.
- Continuing or escalating after nervous laughter or vague discomfort.

3. Offer a gentle check-in if the shift feels abrupt or if future blurring of lines is anticipated.
4. Explicitly invite Joe to say if the new frame isn't working or needs further adjustment.
